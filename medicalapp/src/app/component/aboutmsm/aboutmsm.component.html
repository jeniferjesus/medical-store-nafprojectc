About Us

Medical Store Management website is built for both Medical Staffs as well as Customers to make their lives easy.

Saving data in sheets and hand-written notes can be very time consuming. The biggest advantage in using this website is as follows:

  • Medical Staffs can view & manage medicine information
  • Admin dashboard for staffs to view customer details and orders
  • Customer can login and apply for medicine on the go

Our website does not require or store customer personal information and it stores only their order details. It's a website for customers to order day-to-day medicines. It helps customers to buy medicines regularly from trusted source.
